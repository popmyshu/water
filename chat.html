🔑 首次使用需要 Token

本页面通过 GitHub 同步消息，需要你的
GitHub Personal Access Token

去这里生成：
github.com → 头像 → Settings → Developer settings → Personal access tokens → Generate new token (classic)

需要的权限：repo（全部勾选）

生成后粘贴到下面，点击确认即可。

Token 只保存在你本地浏览器，不会上传到任何第三方
